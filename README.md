# Prueba Técnica

## Descripción del proyecto
Este proyecto corresponde a una prueba técnica enfocada en la implementación de conceptos fundamentales de programación.

El objetivo es demostrar el manejo de estructuras de datos como grafos, colas y pilas, así como el procesamiento y transformación de datos.

Se prioriza la claridad del código, la correcta lógica de implementación y el manejo de casos básicos.

---

## Ejercicios desarrollados

### Ejercicio 1 – Grafos
Se implementa un grafo dirigido utilizando un mapa de adyacencia.
Se aplica un recorrido DFS para determinar si existe un camino entre dos nodos.

### Ejercicio 2 – Cola (FIFO)
Se implementa una cola para procesar comandos respetando el orden FIFO.

### Ejercicio 3 – Pila (LIFO)
Se utiliza una pila para validar si una cadena de paréntesis está balanceada.

### Ejercicio 4 – Procesamiento de datos
Se filtran y transforman datos según condiciones específicas, eliminando información innecesaria.
