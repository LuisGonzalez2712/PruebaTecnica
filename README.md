# PruebaTecnica --> Descripción del proyecto
Este proyecto corresponde a una prueba técnica enfocada en la implementación de conceptos básicos de programación.

El objetivo es demostrar la comprensión y aplicación de estructuras de datos fundamentales como grafos, colas, pilas y también la capacidad de procesar y transformar datos.

A lo largo del proyecto se desarrollaran varios ejercicios, priorizando la legibilidad del código, la correcta lógica de implementación y el manejo de casos básicos.

# Descripción de cada ejercicio
Ejercicio 1 – Grafos

Se modela un grafo dirigido utilizando una estructura de datos tipo mapa.
Se aplica un recorrido (DFS) para determinar si existe un camino entre dos nodos.

Ejercicio 2 – Cola (FIFO)

Se implementa una cola para procesar comandos de entrada y salida.
Se respeta el orden FIFO para obtener la lista de elementos atendidos.

Ejercicio 3 – Pila (LIFO)

Se utiliza una pila para validar si una cadena de paréntesis está balanceada.
Se verifica que cada apertura tenga su cierre correspondiente en el orden correcto.

Ejercicio 4 – Procesamiento de datos

Se filtran elementos según condiciones específicas y se transforman a una nueva estructura.
Además, se eliminan claves innecesarias para mantener los datos limpios.
